# 🌱 typing garden

A website where every key you press grows a tiny plant. Vowels bloom into
flowers, consonants twist into vines, and the spacebar sends a breath ripple
through the air.

## how it works

| key | what grows |
|-----|-----------|
| vowels (a e i o u) | 🌸 a flower that slowly blooms |
| consonants | 🌿 a twisting vine with leaves |
|
